# Personal Portfolio

A responsive personal portfolio website built with HTML, CSS, and JavaScript. It features a modern dark teal-green design, smooth animations, dark/light theme toggle, time-based greeting, and client-side form validation.

## Features

- **Responsive Design** — works on desktop, tablet, and mobile using CSS Grid & Flexbox.
- **Dark / Light Toggle** — theme preference is saved in `localStorage`.
- **Time-Based Greeting** — hero section greeting changes by time of day.
- **Contact Form Validation** — client-side validation with user-friendly error messages.
- **Scroll Animations** — elements fade in as the user scrolls (Intersection Observer).
- **Mobile Navigation** — hamburger menu with smooth slide-in panel.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>
   ```
2. **Open locally** — simply open `index.html` in any modern browser.  
   Alternatively, use a local dev server:
   ```bash
   npx serve .
   ```
3. No build step or dependencies are needed.
